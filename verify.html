casebandit-verify — offline keyless verifier

Runs entirely offline in your browser (no network). It recomputes the audit-ledger chain tip and checks any embedded RFC 3161 token contains that tip's message imprint (a containment check — NOT a full signature/CA validation). For full RFC 3161 signature / CA-chain / genTime validation run the CLI.
